| File | Purpose | Edit? |

| `server.js` | Main backend server | Yes - routes here |

## 🐛 QUICK TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| "Cannot find module" | Run `npm install` |
| "MongoDB connection refused" | Run `mongod` in another window |
| Port 3000 already in use | Change PORT in `.env` |
| No products showing | Run `npm run seed` |
| Changes not reflecting | Restart with `npm run dev` |
| Static files (CSS) not loading | Check files are in `public/` folder |

## 🎯 HOW IT WORKS (30 Second Version)

```
User clicks "Add to Cart"
    ↓
JavaScript calls /api/cart/add endpoint
    ↓
Express server receives request
    ↓
cartController.addToCart() runs
    ↓
Mongoose saves to MongoDB
    ↓
Response sent back to browser
    ↓
Frontend updates UI
    ↓
Cart shows product! ✅
```
